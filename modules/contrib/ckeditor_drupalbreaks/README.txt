INTRODUCTION
------------
This module enables plugin for inserting Drupal teaser and page breaks in your
CKEditor, as it was in 7 version in the CKEditor module or the Wysiwyg module.

INSTALLATION
------------
1. Enable Ckeditor drupalbreaks in the Drupal admin.
2. Configure your WYSIWYG toolbar to include the buttons.

REQUIREMENTS
------------
CKEditor Module (Core)
CKEditor FakeObjects
